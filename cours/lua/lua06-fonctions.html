LUA06 — Fonctions

Débutant 35 min Piste 11 · Lua
PrérequisLUA5 — Boucles.
🎯Objectifs d'apprentissage

À la fin de cette leçon, tu sauras :

  • Déclarer une fonction avec paramètres et return.
  • Renvoyer plusieurs valeurs à la fois.
  • Donner des valeurs par défaut avec or.
  • Comprendre que les fonctions sont des valeurs (first-class) et utiliser ... (varargs).

Pourquoi c'est important

Une fonction regroupe une logique réutilisable : calculer des dégâts, afficher un dialogue, sauvegarder. En Lua, les fonctions sont des valeurs comme les autres — on peut les stocker, les passer en argument, les renvoyer.

Théorie en profondeur

1. Déclaration & retour

local function degats(base, multiplicateur)
  return base * multiplicateur
end

print(degats(10, 1.5))   -- 15.0

Préfère local function pour ne pas polluer l'espace global (rappel de LUA2).

2. Retours multiples

local function minMax(a, b)
  if a < b then return a, b else return b, a end
end

local petit, grand = minMax(8, 3)
print(petit, grand)   -- 3  8

Lua peut renvoyer plusieurs valeurs séparées par des virgules — pratique pour les coordonnées, les bornes, ou un résultat + un message d'erreur.

3. Valeurs par défaut

local function saluer(nom)
  nom = nom or "Inconnu"        -- défaut si nom est nil
  print("Bonjour " .. nom)
end
saluer()         -- Bonjour Inconnu
saluer("Léa")    -- Bonjour Léa

4. Fonctions comme valeurs & varargs

-- une fonction stockée dans une variable
local op = function(a, b) return a + b end
print(op(2, 3))   -- 5

-- nombre variable d'arguments avec ...
local function somme(...)
  local total = 0
  for _, v in ipairs({...}) do total = total + v end
  return total
end
print(somme(1, 2, 3, 4))   -- 10

... capture tous les arguments supplémentaires. On les transforme souvent en table avec {...}.

🔧 Démonstration pas à pas

  1. Écris degats(base, mult) qui renvoie le produit.
  2. Écris minMax(a, b) qui renvoie deux valeurs.
  3. Ajoute une valeur par défaut avec or.
  4. Écris somme(...) avec un nombre variable d'arguments.

🔧Exercice pratique

  1. Écris soigner(pv, montant) qui renvoie les PV plafonnés à 100.
  2. Écris une fonction qui renvoie la moyenne et le total d'une liste.
  3. Écris log(...) qui affiche tous ses arguments.
Critères de réussite
  • Tes fonctions sont déclarées local.
  • Tu exploites au moins un retour multiple.
  • Ta fonction variadique utilise ....

Exercices

Exercice 1Prise en main

Écris une fonction carre(n) qui renvoie n².

Voir une piste 
local function carre(n)
  return n * n
end
print(carre(6))   -- 36
Exercice 2Application

Écris soigner(pv, montant) qui plafonne les PV à 100.

Voir une solution 
local function soigner(pv, montant)
  pv = pv + montant
  if pv > 100 then pv = 100 end
  return pv
end
print(soigner(90, 25))   -- 100
Exercice 3Défi

Écris stats(...) qui renvoie le total et la moyenne des arguments.

Voir une piste 
local function stats(...)
  local args = {...}
  local total = 0
  for _, v in ipairs(args) do total = total + v end
  return total, total / #args
end
local t, m = stats(10, 20, 30)
print(t, m)   -- 60  20.0
📌À retenir
  • local function nom(params) … return … end.
  • Lua sait renvoyer plusieurs valeurs : return a, b.
  • Valeur par défaut : p = p or défaut (sauf si false est valide).
  • Les fonctions sont des valeurs ; ... capture les varargs.

Glossaire des termes introduits

first-class function — une fonction qui peut être stockée, passée et renvoyée comme n'importe quelle valeur.

varargs (...) — nombre variable d'arguments.

retours multiples — capacité de renvoyer plusieurs valeurs d'un coup.

Voir aussi le glossaire global.
