LUA01 — Introduction à Lua

Débutant 35 min Piste 11 · Lua
Prérequisaucune expérience requise. Utile : avoir vu un autre langage (C#, GDScript) aide, mais ce n'est pas nécessaire.
🎯Objectifs d'apprentissage

À la fin de cette leçon, tu sauras :

  • Ce qu'est Lua et pourquoi c'est un langage embarqué léger.
  • Où Lua est utilisé dans le jeu (Roblox/Luau, LÖVE, Defold…).
  • Écrire et exécuter un premier script (print).
  • Reconnaître la syntaxe de base (commentaires, instructions).

Pourquoi c'est important

Lua est partout dans le jeu vidéo : c'est le langage de Roblox (sous le nom Luau), du moteur LÖVE (love2d), de Defold, et il sert de langage de script embarqué dans d'innombrables jeux (mods, configuration, logique). Léger, rapide et simple, il s'apprend très vite — un excellent 4ᵉ langage après C#, C++ et GDScript.

Théorie en profondeur

1. Un langage de script embarqué

Lua n'est (presque) jamais utilisé seul : il est embarqué dans un programme hôte (un moteur de jeu) écrit en C/C++. L'hôte expose des fonctions (déplacer un objet, jouer un son), et tu écris la logique en Lua. Cette séparation — moteur rapide en C, logique souple en Lua — est très répandue car elle combine performance et flexibilité (on modifie le Lua sans recompiler le moteur).

2. Où on le rencontre

  • Roblox : utilise Luau (un Lua étendu, typé en option). Des millions de jeux y sont scriptés en Luau.
  • LÖVE (love2d) : framework 2D où tout se code en Lua (love.load, love.update, love.draw).
  • Defold : moteur 2D/3D scripté en Lua.
  • Et de nombreux jeux/outils l'embarquent pour les mods et la configuration.

3. Premier script

L'instruction de base : print, qui affiche dans la console/sortie :

-- ceci est un commentaire (deux tirets)
print("Bonjour Lua !")        -- affiche : Bonjour Lua !

--[[ commentaire
     sur plusieurs lignes ]]
local message = "Salut"
print(message)

Pas de point-virgule obligatoire, pas d'accolades pour les blocs (on utilise des mots-clés comme end, vu plus loin). Les commentaires : -- (ligne) et --[[ ]] (bloc).

4. Comment l'exécuter

Selon le contexte : dans Roblox Studio (un Script dans l'explorateur), dans un projet LÖVE (main.lua), ou via l'interpréteur lua en ligne de commande pour s'entraîner. Lua est interprété : pas de compilation longue, itération immédiate.

🔧 Démonstration pas à pas

  1. Choisis un environnement : Roblox Studio, un projet LÖVE, ou l'interpréteur lua en ligne.
  2. Écris print("Bonjour Lua !") et exécute.
  3. Ajoute un commentaire de ligne (--) et un bloc (--[[ ]]).
  4. Stocke un texte dans une variable local et affiche-la.

⚠️ Pièges & erreurs courantes

SymptômeMon print n'affiche rien.
CauseLa sortie/console n'est pas ouverte (Output dans Roblox, terminal pour LÖVE).
CorrectionOuvre la fenêtre Output/console de ton environnement.
Symptôme« unexpected symbol » au lancement.
CauseSyntaxe d'un autre langage (accolades, point-virgule obligatoire…).
CorrectionEn Lua : pas d'accolades pour les blocs, commentaires en --.
SymptômeMon script Roblox ne s'exécute pas.
CauseMauvais type de script ou mauvais emplacement.
CorrectionPlace un Script au bon endroit (renvoi LUA13).
SymptômeConfusion Lua / Luau.
CauseLuau (Roblox) ajoute le typage optionnel et des extensions.
CorrectionLes bases sont identiques ; on les apprend, Luau ajoute des options par-dessus.

🔧Exercice pratique

  1. Affiche ton prénom puis une seconde ligne avec un autre print.
  2. Ajoute un commentaire de ligne et un commentaire de bloc.
  3. Stocke un nombre et un texte dans des variables et affiche-les.
Critères de réussite
  • Les deux lignes s'affichent dans la sortie.
  • Tes commentaires n'apparaissent pas à l'exécution.
  • Tu sais relancer ton script rapidement.

Glossaire des termes introduits

Lua — langage de script léger, conçu pour être embarqué dans un programme hôte.

Embarqué (embedded) — intégré à un moteur/programme qui l'exécute et lui expose une API.

Luau — variante de Lua utilisée par Roblox (typage optionnel, extensions).

print — affiche un message dans la sortie/console.
