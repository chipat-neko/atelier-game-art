G6+ — TP de synthèse : intégrer la caisse

Intermédiaire 40 min Piste 5 · Godot
PrérequisG5 — Import, G6 — Matériaux et L2c — Bake & export.
🎯Objectif du TP

Boucler le pipeline dans Godot : importer la caisse en glTF, reconstruire son StandardMaterial3D (albedo, normal, metallic, roughness, AO), éclairer et comparer au rendu de référence ref_high.png.

📦Fichiers de l'exercice

La caisse — le .glb s'importe directement (mesh + UV, et un matériau de base). Pour le rendu final, branche les maps de L2c.

  • caisse_low.glb
  • .fbx
  • UV
  • README

Cible visuelle : ref_high.png. Godot lit des maps séparées (ou une ORM via canaux).

🔧 Démonstration pas à pas

  1. Importer. Glisse caisse_low.glb dans le FileSystem : Godot l'importe (mesh + UV). Glisse-le dans la scène pour l'instancier. Le .glb porte déjà un matériau basique — bon point de départ.
  2. Créer le matériau. Sélectionne le MeshInstance3D ▸ Inspector ▸ Surface Material Override ▸ New StandardMaterial3D. Déplie-le.
  3. Brancher les maps. Albedo ▸ Texture = ta Base Color. Section Normal Map ▸ coche Enabled + charge la Normal. Metallic ▸ Texture + règle le canal (Blue si tu utilises une ORM). Roughness ▸ Texture + canal (Green pour ORM). Ambient Occlusion ▸ coche Enabled + Texture + canal (Red pour ORM).
  4. Vérifier les imports. Sélectionne la Base Color dans le FileSystem ▸ onglet Import : elle est en sRGB. Les maps de données (Normal/Metallic/Roughness/AO) : sRGB désactivé (Godot le gère souvent automatiquement selon l'usage).
  5. Éclairer. Ajoute une DirectionalLight3D (ombres activées) et un WorldEnvironment (ciel + ambiance, voir G7). Oriente le soleil pour révéler la Normal.
  6. Comparer. Mets ref_high.png à côté du viewport : matières, usure, reflets. Ajuste la Roughness et l'angle du soleil.
💡Une ORM dans Godot

Si tu as exporté une ORM (comme pour Unreal), tu peux la réutiliser dans Godot : charge la même texture ORM dans Metallic, Roughness et AO, et règle le canal de chacun (Metallic = Blue, Roughness = Green, AO = Red). Un seul fichier alimente trois canaux.

🖐Essaie maintenant~1 min

Dans le StandardMaterial3D, décoche Normal Map ▸ Enabled puis recoche : la surface passe de lisse à « planches marquées ». Puis pousse la Metallic à 1 sur tout l'objet : observe comment la couleur cède la place aux reflets. Remets la texture ensuite.

⚠️ Pièges & erreurs courantes

Symptôme La Normal ne fait rien.
Cause Section Normal Map non Enabled, ou texture importée en Normal Map non signalée.
Correction Coche Enabled sous Normal Map ; vérifie l'import (onglet Import ▸ Normal Map).
Symptôme Metallic/Roughness donnent un résultat inversé ou faux.
Cause Mauvais canal choisi (Red/Green/Blue) pour une ORM.
Correction Metallic = Blue, Roughness = Green, AO = Red (packing ORM).
Symptôme Couleurs délavées.
Cause Base Color importée en linéaire, ou données en sRGB.
Correction Base Color en sRGB ; maps de données sans sRGB (onglet Import).
Symptôme Zones d'ombre anormalement noires.
Cause Pas de WorldEnvironment (ni ciel ni ambiance).
Correction Ajoute un WorldEnvironment avec un ciel pour la lumière ambiante et les reflets.

🔧Exercice pratique

Intègre la caisse texturée dans Godot et rapproche-toi de ref_high.png.

  1. Importe le .glb et instancie-le dans une scène.
  2. Construis un StandardMaterial3D avec Albedo, Normal (Enabled), Metallic, Roughness et AO (bons canaux si ORM).
  3. Éclaire avec DirectionalLight3D + WorldEnvironment.
  4. Compare à la référence et ajuste.
Critères de réussite
  • Le matériau utilise Albedo + Normal (Enabled) + Metallic + Roughness + AO.
  • Si ORM : les canaux sont corrects (Metallic=Blue, Roughness=Green, AO=Red).
  • Base Color en sRGB, maps de données en linéaire.
  • La scène a un WorldEnvironment (pas d'ombres 100 % noires).
  • Le rendu Godot ressemble à ref_high.png.

Vérifie ta compréhension

Clique sur la réponse que tu penses correcte : la bonne réponse et une explication s'affichent aussitôt.

📌À retenir
  • On reconstruit un StandardMaterial3D : albedo + normal (Enabled) + metallic + roughness + AO.
  • Une ORM se réutilise par canaux : Metallic=Blue, Roughness=Green, AO=Red.
  • Base Color en sRGB, données en linéaire (onglet Import).
  • Un WorldEnvironment évite les ombres 100 % noires et donne les reflets.

Pour aller plus loin

  • L2c : refaire l'export si les maps posent problème.
