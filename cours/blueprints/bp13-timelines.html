BP13 — Timelines & animation par BP

Avancé 40 min Piste 12 · Blueprints
PrérequisBP5 — Événements, BP11 — Composants.
🎯Objectifs d'apprentissage

À la fin de cette leçon, tu sauras :

  • Ce qu'est un nœud Timeline et ses sorties (Update, Finished).
  • Utiliser des tracks Float / Vector / Event.
  • Interpoler avec Lerp pour animer une transform.
  • Animer une porte/plateforme sans code, avec Play / Reverse.

Pourquoi c'est important

Beaucoup d'animations de gameplay (porte qui s'ouvre, plateforme qui monte, fondu) n'ont pas besoin d'un logiciel d'animation : une Timeline dans le Blueprint suffit et reste pilotable par la logique.

Théorie en profondeur

1. Le nœud Timeline

Une Timeline joue une courbe dans le temps. Entrées d'exécution : Play, Play from Start, Reverse, Stop. Sorties : Update (à chaque frame de lecture) et Finished (à la fin). Une broche de données fournit la valeur courante de chaque track.

2. Les tracks

  • Float Track : une valeur 0→1 (souvent un alpha d'interpolation).
  • Vector Track : trois valeurs (ex. une position).
  • Event Track : déclenche un événement à un instant précis.

3. Animer avec Lerp

On utilise l'alpha de la Float Track pour interpoler (Lerp) entre un état A et un état B. Sur Update, on applique la valeur interpolée à la transform.

Timeline "OuvrirPorte" (Float 0→1 sur 1s)
   Update ─exec▶ Set Relative Rotation =
                 Lerp(RotFermee, RotOuverte, Alpha)
   Finished ─exec▶ Set bOuverte = true

4. Play / Reverse (ouvrir/fermer)

Le même Timeline sert dans les deux sens : Play ouvre, Reverse ferme. Couplé à un FlipFlop (vu en BP7), on bascule l'état à chaque interaction.

🔧 Démonstration pas à pas

  1. Ajoute un Timeline avec une Float Track 0→1 sur 1 s.
  2. Sur Update, Lerp entre rotation fermée et ouverte → Set Relative Rotation.
  3. Sur Finished, mets à jour l'état.
  4. Branche Play / Reverse via un FlipFlop sur l'interaction.

⚠️ Pièges & erreurs courantes

SymptômeLa porte saute d'un coup.
CauseSet fait en une fois, pas sur Update.
CorrectionApplique la valeur sur Update chaque frame.
SymptômeAnimation qui ne rejoue pas.
CausePlay reprend là où on s'était arrêté.
CorrectionUtilise Play from Start pour repartir de zéro.
SymptômeLerp dans le mauvais sens.
CauseA et B inversés.
CorrectionVérifie l'ordre (A = départ, B = arrivée) et l'alpha.
SymptômeTimeline introuvable dans une fonction.
CauseLes Timelines sont des nœuds latents.
CorrectionPlace-les dans l'Event Graph, pas dans une fonction.

🔧Exercice pratique

  1. Anime une plateforme qui monte de 0 à 200 en Z (Lerp sur Update).
  2. Ajoute un Event Track qui joue un son à mi-parcours.
  3. Ouvre/ferme avec Play / Reverse via FlipFlop.
Critères de réussite
  • Le mouvement est fluide (appliqué sur Update).
  • Play/Reverse fonctionnent dans les deux sens.
  • Tu places la Timeline dans l'Event Graph.

Exercices

Exercice 1Prise en main

Quelle sortie du Timeline utiliser pour appliquer le mouvement ?

Voir une solution

Update (chaque frame). Finished sert à conclure (changer l'état, jouer un son de fin).

Exercice 2Application

Décris l'animation d'ouverture de porte (rotation).

Voir une piste 
Play ─▶ Timeline (Float 0→1)
Update ─▶ Set Relative Rotation = Lerp(0°, 90°, Alpha) (autour de Z)
Exercice 3Défi

Conçois une plateforme qui fait des allers-retours en continu.

Voir une piste

Sur Finished, rappelle Reverse (et inversement) — ou active la lecture en boucle de la Timeline (option Loop) avec une courbe qui monte puis redescend (0→1→0). Le mouvement reste appliqué sur Update.

📌À retenir
  • Le Timeline joue une courbe dans le temps : Update (chaque frame), Finished (fin).
  • Tracks Float / Vector / Event.
  • Anime avec Lerp(A, B, Alpha) appliqué sur Update.
  • Play / Reverse pour ouvrir/fermer ; Timeline dans l'Event Graph.

Glossaire des termes introduits

Timeline — nœud qui joue une courbe dans le temps (latent).

Track — piste d'une Timeline (Float, Vector ou Event).

Lerp (interpolation linéaire) — mélange A→B selon un alpha 0→1.

Update / Finished — sorties de la Timeline (chaque frame / fin).
