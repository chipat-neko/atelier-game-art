SH10 — Effets courants (dissolve, rim…)

Avancé 45 min Piste 10 · Shaders
PrérequisSH6, SH7 et SH9 — Fresnel.
🎯Objectifs d'apprentissage

À la fin de cette leçon, tu sauras :

  • Créer un effet de dissolve (seuil sur une texture de bruit).
  • Réaliser un contour Fresnel (rim light).
  • Composer un hologramme (lignes + Fresnel + transparence).
  • Réutiliser step/smoothstep/lerp/Fresnel pour combiner des effets.

Pourquoi c'est important

Les effets « signature » (dissolution d'un ennemi, halo de bouclier, hologramme d'UI) reposent sur peu d'ingrédients combinés intelligemment. Savoir les construire, c'est maîtriser le vocabulaire (seuil, masque, Fresnel, alpha) qu'on remixe à l'infini.

Théorie en profondeur

1. Dissolve

On échantillonne une texture de bruit (niveaux de gris). On compare au seuil animé : au-dessus, le pixel disparaît (alpha 0) ; sinon il reste. Une fine bande autour du seuil s'illumine (le « bord ardent »).

float bruit = tex2D(_NoiseMap, uv).r;
clip(bruit - _Seuil);                       // jette le pixel si bruit < seuil
float bord = smoothstep(_Seuil, _Seuil + 0.05, bruit);  // 0 sur le bord
float3 couleur = lerp(_CouleurBord, albedo, bord);      // bord lumineux

2. Rim light (contour Fresnel)

Le Fresnel (SH9) éclaire les bords vus de profil : idéal pour détacher un personnage du fond ou simuler un bouclier.

float rim = pow(1.0 - saturate(dot(N, V)), _Puissance);
float3 couleur = albedo + _CouleurRim * rim;

3. Hologramme

On combine : des lignes de balayage (sinus sur la position écran/UV + frac), un Fresnel bleuté, et une transparence (blending). On peut faire défiler les lignes avec le temps. C'est l'archétype de l'effet composé : chaque ingrédient est simple, le mélange fait l'effet.

🔧 Démonstration pas à pas

  1. Branche une texture de bruit, ajoute un paramètre Seuil et utilise clip pour faire disparaître les pixels.
  2. Anime le seuil dans le temps (0→1) : l'objet se dissout ; ajoute un bord lumineux avec smoothstep.
  3. Ajoute un rim Fresnel et règle sa puissance/couleur.
  4. Compose un hologramme : lignes (sinus + frac), Fresnel bleuté, transparence.

⚠️ Pièges & erreurs courantes

SymptômeMon dissolve fait des trous nets sans bord.
CauseTu utilises seulement clip, sans bande de bord.
CorrectionAjoute un smoothstep autour du seuil pour le bord ardent.
SymptômeMon rim illumine toute la surface.
CausePuissance de Fresnel trop faible.
CorrectionAugmente l'exposant pour concentrer l'effet sur les bords.
SymptômeL'hologramme est opaque.
CauseLe matériau n'est pas en mode transparent.
CorrectionActive le blending (rappel SH2).
SymptômeEffet correct mais perfs en chute.
CauseTrop de samples/calculs lourds par pixel.
CorrectionSimplifie le fragment (voir SH14).

🔧Exercice pratique

  1. Réalise un dissolve animé avec bord lumineux.
  2. Ajoute un rim light Fresnel réglable.
  3. Compose un hologramme (lignes + Fresnel + transparence).
Critères de réussite
  • Le dissolve disparaît progressivement avec un bord net.
  • Le rim ne touche que les contours.
  • L'hologramme est translucide et animé.

Exercices

Exercice 1Prise en main

Crée un rim light simple et règle sa couleur/puissance pour détacher un objet du fond.

Voir une piste

rim = pow(1 - saturate(dot(N, V)), puissance); couleur = albedo + couleurRim * rim. Augmente la puissance pour un liseré fin, baisse-la pour un halo plus large.

Exercice 2Application

Réalise un dissolve animé (seuil piloté par le temps) avec un bord ardent.

Voir une piste

clip(bruit - seuil) pour effacer ; bord = smoothstep(seuil, seuil+largeur, bruit) ; couleur = lerp(couleurBord, albedo, bord). Anime « seuil » de 0 à 1 pour la dissolution.

Exercice 3Défi

Compose un hologramme : lignes de balayage animées + Fresnel bleuté + transparence.

Voir une piste

lignes = smoothstep(0.4, 0.5, frac(posY * densite - temps)) ; fresnel pour les bords ; alpha = base + lignes*0.2 + fresnel ; matériau transparent. Chaque morceau est simple : c'est leur combinaison qui crée l'hologramme.

📌À retenir
  • Dissolve : seuil animé sur une texture de bruit (clip) + bord via smoothstep.
  • Rim light : Fresnel (pow(1 - dot(N, V), p)) ajouté à la couleur.
  • Hologramme : lignes (sinus/frac) + Fresnel + transparence.
  • Les effets se construisent en combinant peu d'ingrédients (seuil, masque, Fresnel, alpha).

Glossaire des termes introduits

Dissolve — effet d'apparition/disparition par seuil sur un bruit.

clip / discard — instruction qui rejette le pixel courant (transparence dure).

Rim light — halo de contour basé sur le Fresnel.

Bruit (noise) — texture pseudo-aléatoire pilotant des effets organiques.
