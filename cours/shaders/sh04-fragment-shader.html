SH04 — Le fragment (pixel) shader

Intermédiaire 40 min Piste 10 · Shaders
PrérequisSH3 — Vertex shader.
🎯Objectifs d'apprentissage

À la fin de cette leçon, tu sauras :

  • Le rôle du fragment shader : produire la couleur (RGBA) de chaque pixel.
  • Utiliser les entrées interpolées (UV, normale) venues du vertex.
  • Créer un dégradé à partir des UV.
  • Comprendre la sortie couleur et le canal alpha.

Pourquoi c'est important

Le fragment shader est là où se joue presque tout le « look » : couleur, textures, éclairage, effets. Apprendre à lire ses entrées et à composer sa sortie est la compétence centrale de la programmation graphique.

Théorie en profondeur

1. Entrées : des valeurs interpolées par pixel

Le fragment reçoit les sorties du vertex, interpolées pour le pixel courant (rappel SH2) : l'UV exact, la normale, la position monde… À partir de ces valeurs, il calcule une couleur.

2. La sortie : une couleur RGBA

Le fragment renvoie un vecteur de 4 composantes (r, g, b, a), chacune entre 0 et 1. a (alpha) sert à la transparence (si le blending est actif).

// Dégradé horizontal à partir de l'UV
float4 frag(float2 uv : TEXCOORD0) : SV_Target
{
    float intensite = uv.x;                 // 0 à gauche, 1 à droite
    return float4(intensite, intensite, intensite, 1.0);  // niveaux de gris
}

3. Premiers dégradés avec les UV

Les UV varient de 0 à 1 sur la surface (vu en F6). En les utilisant comme couleur, on obtient des dégradés : uv.x horizontal, uv.y vertical, la distance au centre pour un cercle. C'est l'exercice fondateur pour « penser par pixel ».

4. Composer une couleur

On combine des sources : une couleur de base, une texture (échantillonnée, vu en SH6), un masque, l'éclairage (vu en SH8). Le fragment est un mini-programme de calcul de couleur : on l'enrichit progressivement.

🔧 Démonstration pas à pas

  1. Renvoie uv.x comme niveau de gris : tu obtiens un dégradé horizontal.
  2. Mets uv directement dans (R,G) : tu vois le « damier de couleurs » UV (rouge→droite, vert→haut).
  3. Calcule la distance de l'UV au centre (0.5, 0.5) et seuil-la pour dessiner un disque.
  4. Multiplie ta couleur par une couleur de base réglable (paramètre du matériau).

⚠️ Pièges & erreurs courantes

SymptômeCouleurs « cramées » ou ternes.
CauseValeurs hors [0,1] ou confusion sRGB/linéaire (voir F1).
CorrectionGarde les couleurs dans [0,1] et respecte l'espace colorimétrique.
SymptômeMon dégradé est uniforme (pas de variation).
CauseL'UV n'est pas transmis depuis le vertex, ou vaut une constante.
CorrectionVérifie que le vertex passe bien l'UV (varying).
SymptômeMa transparence ne marche pas.
CauseL'alpha est calculé mais le matériau est opaque (blending off).
CorrectionPasse le matériau en mode transparent (rappel SH2).
SymptômeLe shader est lent.
CauseCalcul lourd dans le fragment (exécuté par pixel).
CorrectionAllège le fragment, déporte ce qui peut l'être (voir SH14).

🔧Exercice pratique

  1. Crée un dégradé horizontal puis vertical à partir des UV.
  2. Dessine un disque (distance au centre + seuil).
  3. Ajoute une couleur de base réglable multipliée à ta sortie.
Critères de réussite
  • Les dégradés suivent bien u et v.
  • Le disque est net (seuil) ou doux (smoothstep).
  • La couleur de base modifie le rendu.

Exercices

Exercice 1Prise en main

Affiche les UV comme couleur (R=u, G=v) pour visualiser le « plan » de texture de l'objet.

Voir une piste

return float4(uv.x, uv.y, 0, 1). Tu vois un dégradé rouge vers la droite et vert vers le haut : c'est une excellente façon de vérifier que les UV de l'objet sont corrects.

Exercice 2Application

Dessine un disque centré : calcule la distance de l'UV à (0.5, 0.5) et applique un seuil.

Voir une piste

float d = distance(uv, float2(0.5, 0.5)); float cercle = step(d, 0.3); — cercle vaut 1 à l'intérieur du rayon 0.3, 0 dehors. Remplace step par smoothstep pour un bord doux (anti-aliasé).

Exercice 3Défi

Combine une couleur de base, un dégradé et un masque (disque) en une seule sortie cohérente.

Voir une piste

Calcule un dégradé, multiplie par une couleur de base, puis utilise le masque disque comme alpha (ou pour mélanger deux couleurs avec lerp). Tu composes ainsi un « badge » coloré : la logique de tout effet plus complexe.

📌À retenir
  • Le fragment shader produit une couleur RGBA par pixel.
  • Ses entrées (UV, normale…) viennent du vertex et sont interpolées.
  • Les UV (0→1) servent à fabriquer dégradés, masques et formes.
  • On compose la couleur en combinant base, textures, masques et éclairage.

Glossaire des termes introduits

Fragment / pixel shader — programme calculant la couleur d'un pixel.

RGBA — composantes rouge, vert, bleu, alpha (transparence) d'une couleur.

UV — coordonnées de texture (0→1) servant à situer un point sur la surface.

step / smoothstep — seuil net / transition douce (vu en SH7).
