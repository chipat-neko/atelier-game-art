U6+ — TP de synthèse : intégrer la caisse

Intermédiaire 40 min Piste 3 · Unity
PrérequisU5 — Import, U6 — Matériaux et L2c — Bake & export (pour tes maps).
🎯Objectif du TP

Boucler le pipeline côté moteur : importer la caisse et ses maps PBR, reconstruire son matériau en URP, éclairer une petite scène et comparer au rendu de référence ref_high.png. C'est le point où tout ce que tu as fait en amont (modélisation, UV, texturing, export) se vérifie enfin dans le moteur.

📦Fichiers de l'exercice

La caisse — importe le mesh, et utilise les maps que tu as exportées en L2c (preset Unity URP). Pas encore fait ? Le .glb embarque déjà un matériau de base pour dépanner.

  • caisse_low.fbx
  • .glb
  • UV
  • README

Cible visuelle : ref_high.png. Maps attendues : Base Color, Normal, Metallic (Smoothness en alpha), Occlusion.

🔧 Démonstration pas à pas

  1. Importer le mesh et les maps. Glisse caisse_low.fbx et tes textures dans la fenêtre Project. Dans les Import Settings du FBX, règle le Scale Factor si besoin (la caisse doit faire ~1 m).
  2. Régler les espaces couleur. Sélectionne la Base Color : sRGB coché. La Normal : passe son Texture Type sur Normal map. La Metallic et l'Occlusion : sRGB décoché (ce sont des données, pas des couleurs). C'est l'erreur n°1 des imports.
  3. Créer le matériau. Clic droit dans Project ▸ Create ▸ Material, shader Universal Render Pipeline/Lit. Assigne : Base Map, Normal Map, Occlusion Map, et Metallic Map avec Smoothness Source = Metallic Alpha.
  4. Poser dans la scène. Glisse le mesh dans la vue Scene, puis glisse le matériau dessus. La caisse prend sa matière.
  5. Éclairer. Assure-toi d'avoir une Directional Light (le « soleil ») avec les ombres activées, et un skybox/environnement pour la lumière ambiante et les reflets (voir U7). Oriente la lumière pour révéler le relief de la Normal.
  6. Comparer. Place ref_high.png à côté de la vue Game : mêmes matières, même usure, même intensité de reflets ? Ajuste le Smoothness et l'angle de lumière.

🖐Essaie maintenant~1 min

Sur le matériau, décoche puis recoche la Normal Map : sans elle, la caisse redevient lisse ; avec elle, planches et joints réapparaissent. Tu vois exactement ce que le faux relief apporte, sans un seul polygone en plus.

⚠️ Pièges & erreurs courantes

Symptôme L'éclairage de la caisse semble « plat » ou faux.
Cause La Normal n'est pas déclarée comme Normal map (Unity la lit comme une couleur).
Correction Texture Type ▸ Normal map ▸ Apply. Unity propose souvent « Fix now ».
Symptôme Couleurs délavées / rugosité fausse.
Cause sRGB coché sur Metallic/Occlusion (des données).
Correction Décoche sRGB sur toutes les maps sauf Base Color.
Symptôme Le mat paraît brillant, le brillant paraît mat.
Cause Smoothness Source mal réglée ou map non inversée à l'export.
Correction Smoothness Source = Metallic Alpha ; exporte depuis Painter avec le preset URP.
Symptôme La caisse est minuscule ou géante.
Cause Échelle non appliquée avant export, ou Scale Factor d'import.
Correction Applique l'échelle dans Blender (Ctrl A) et ajuste le Scale Factor à l'import.

🔧Exercice pratique

Intègre la caisse texturée dans Unity et rapproche-toi de ref_high.png.

  1. Importe le mesh et tes maps L2c avec les bons espaces couleur.
  2. Construis un matériau URP/Lit (Base, Normal, Metallic+Smoothness, Occlusion).
  3. Éclaire avec une Directional Light + skybox ; active les ombres.
  4. Compare à la référence et ajuste Smoothness + lumière.
Critères de réussite
  • Les maps sont importées avec les bons espaces couleur (Normal en Normal map).
  • Le matériau URP/Lit montre bois + métal aux bons endroits, avec l'usure.
  • Le relief (Normal/Height) est visible sous un éclairage rasant.
  • Le rendu Unity ressemble à ref_high.png.
  • Tu sais expliquer le rôle de la Smoothness (= 1 − Roughness) dans Unity.

Vérifie ta compréhension

Clique sur la réponse que tu penses correcte : la bonne réponse et une explication s'affichent aussitôt.

📌À retenir
  • Un asset ne « marche » vraiment qu'une fois intégré et éclairé dans le moteur.
  • Normal map = Texture Type Normal map ; données (Metallic/Occlusion) = sRGB décoché.
  • URP range la Smoothness dans l'alpha de la Metallic (= 1 − Roughness).
  • La référence guide les réglages : compare et ajuste Smoothness + lumière.
